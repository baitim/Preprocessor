<h1 align="center">ASM & SPU & DISASM</h1>

## Description

 Implementation of assembler, processor and disassembler.

## How to use

1. Clone <br>
        write <code>git clone https://github.com/baitim/Stack.git</code> in terminal

2. Go to folder <br>
        write <code>cd Preprocessor</code> in terminal

3. Build <br>
        write <code>make</code> in terminal

4. Run <br>
        write <code>.\start.exe</code> in termina

## Feats

* This project can assemble disassemble and process code

* Jumps are supported

* Registers are supported

* Labels are supported

* Functions is supported

* Ram is supported

* Implemented functions:
    * PUSH <code> < register | num | ram > </code>
    * IN
    * POP <code> < register | ram > </code>
    * ADD
    * SUB
    * MUL
    * DIV
    * SQRT
    * SIN
    * COS
    * OUT
    * HLT
    * JMP <code> < lable | num > </code>
    * JA <code> < lable | num > </code>
    * JAE <code> < lable | num > </code>
    * JB <code> < lable | num > </code>
    * JBE <code> < lable | num > </code>
    * JE <code> < lable | num > </code>
    * JNE <code> < lable | num > </code>
    * JMON <code> < lable | num > </code>
    * CALL <code> < lable > </code>
    * RET
    * OUTC
